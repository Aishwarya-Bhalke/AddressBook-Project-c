# 📒 Address Book Management System in C

A console-based **Address Book Management System** developed in **C** using **File Handling, Structures, Functions, and Modular Programming**.

---

## 🚀 Features

- Add new contacts
- Search contacts
- Edit contacts
- Delete contacts
- View all contacts
- Store contact details using file handling
- Input validation for phone numbers and email addresses
